# Штатаная книга

## Описание проекта
Настольное Python-приложение для работы со штатной книгой с функцией анализа и визуализации данных

## Функционал
- Добавление, редактирование, удаление позиций штатного расписания;
- Выгрузка данных в excel;
- Анализ;
- Визуализация данных.

## Запуск
```
pip install matplotlib, pandas, psycopg2, tk, seaborn
python main.app

```
