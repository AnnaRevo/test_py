# Инструкция по работе с командами Git
## 1: Создание локального репозитория с помощью функции *git init*
## 2: Получение информации о текущем состоянии репозитория с помощью функции *git status*
## 3: Добавление файлов к следующему коммиту с помощью функции *git add*
## 4: Создание коммита с помощью функции *git commit -m "message"*
## 5: Выведение истории коммитов с помощью функции *git log*
